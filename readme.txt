游戏：微微麻将
简介：以最基础的麻将规则为主体，采用一种花色进行两人麻将。
网络贴图素材来自网络，需要使用该项目请自行替换游戏的的图片资源等，该项目免费开源，该项目未经过大量测试，还有很多bug，可提供学习和参考。有发现的bug可向邮箱中提出，后续修复。
源码：前端采用Cocos Creator版本 v1.5.1，后台采用node.js 版本v8.1.4
前端目录结构：
	Scene:	场景文件
	Script: 脚本文件
	image: 图片文件
	perfab: 预制文件
	resources: 资源文件
后台目录结构：
	majiang：麻将后台源文件
	mode_modules：node.js所用到的框架
	majiang.bat：批处理文件，用于启动服务器

主要内容：
	1游戏大体界面布局
	2游戏逻辑
	3游戏后台基础功能
未完成内容：
